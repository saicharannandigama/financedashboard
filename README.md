# 💰 Finance Dashboard

## 🚀 Overview
A clean and interactive finance dashboard to track income, expenses, and spending patterns.

## ✨ Features
- Dashboard summary (Balance, Income, Expenses)
- Interactive charts (Spending Overview)
- Transactions list
- Role-based UI (Admin / Viewer)
- Insights section (Top spending category)

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts

## ▶️ Run Locally
npm install  
npm run dev
